# Contacts Manager - Java Project

## Description
Ce projet est une application simple en Java qui permet de gérer une liste de contacts.

On peut :
- Ajouter des contacts (nom + numéro de téléphone)
- Rechercher un contact par son nom
- Afficher le numéro de téléphone d’un contact

---

## Objectif de l’exercice
Apprendre les bases de la programmation orientée objet en Java :
- Création de classes
- Utilisation d’objets
- Tableaux
- Méthodes
- Recherche dans une liste
